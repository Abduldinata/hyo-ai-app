# Expression Assets

Tambahkan 4 file PNG untuk ekspresi Hyo AI:

- **idle.png** - Ekspresi netral/idle
- **happy.png** - Ekspresi senang/gembira
- **sad.png** - Ekspresi sedih
- **angry.png** - Ekspresi marah

## Ukuran yang disarankan:
- 512x512 px atau 1024x1024 px
- Format: PNG dengan background transparan
- Style: Anime character portrait

Sampai file ditambahkan, app akan menampilkan placeholder icon.
